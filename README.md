Costa Norte Group
Costa Norte Group es una plataforma inmobiliaria. El sistema ofrece una experiencia de usuario minimalista y sofisticada, enfocada en resaltar la arquitectura y exclusividad de las propiedades disponibles para venta y alquiler.

🏛️ Características Principales
UI/UX Editorial: Diseño inspirado en revistas de arquitectura moderna, con un enfoque en la legibilidad y el minimalismo.

Directorio Dinámico: Filtrado inteligente de propiedades por ubicación, tipo de operación y características.

Framer Motion Integration: Transiciones fluidas y micro-interacciones que elevan la percepción de calidad.

Panel Administrativo: Gestión privada de publicaciones (CRUD) para agentes inmobiliarios.

Responsive Design: Optimización total para dispositivos móviles, garantizando una navegación fluida en cualquier pantalla.

🚀 Stack Tecnológico
Frontend
React.jsx (Vite)

Tailwind CSS (Estilos y diseño responsive)

Framer Motion (Animaciones de interfaz)

Lucide React (Iconografía minimalista)

React Router Dom (Navegación SPA)
